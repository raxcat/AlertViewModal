# AlertViewModal

Inspirated by UIAlertViewController and UIAlertView in iOS. Based on Marionette, bootstrap, underscore. 

 See [JSFiddle] for a live demo.

### Dependency
- bootstrap.js
- bootstrap.css
- jquery.js
- underscore.js
- backbone.js
- backbone.marionette.js
